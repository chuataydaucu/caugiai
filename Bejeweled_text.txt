Then, the gems from the top left corner start. Each byte represents one gem, starting from the first line until the very last one. Depending on what gem we have, it might be stored in more than one bytes.
An empty gem (FF) is stored in only one byte. 
A normal gem is stored in 2 bytes. The first byte is the skin color and the second byte represents if itâ€™s powered (01) or not (00). The skin colors are shown in the picture starting from 00 until 09. 
Lastly, a gem with skin value of 07 is a bomb and takes 3 bytes instead of 2. The reason for that is that we need to know the value of the bomb on the third byte (in the image, that value is 09, so the bomb will have the value 9).
